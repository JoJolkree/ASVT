MBR + пустые сектора + описание файлов.
На каждый файл по три трека.
Описание файла: 2Б - наличие файла, 2Б - размер файла (в байтах), имя файла - 30Б

Номера файлов - от 0 до 12
Название программы - первая локальная переменная

С адреса 4096 пишем описание файла
По нужному адресу - сам файл

Для open():
O_RDONLY    = 0
O_WRONLY    = 1
O_RDWR      = 2
